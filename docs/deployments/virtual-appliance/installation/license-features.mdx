# Licensing features

Some Appliance features must be specifically licensed. Each feature is identified by a short character code, described in the table below:

| Feature code | Description                                                                                       |
| ------------ | ------------------------------------------------------------------------------------------------- |
| MAPBA        | Batch transcription                                                                               |
| MAPRT        | Realtime transcription [^1]                                                                       |
| APR          | Custom dictionary                                                                                 |
| ASS          | Speaker change                                                                                    |
| APD          | Speaker diarization                                                                               |
| AOP2         | Enhanced operating point                                                                          |
| ALID         | Language identification                                                                           |
| ATRA         | Translation [^1]                                                                                  |
| LANY         | Transcription in any language                                                                     |
| L\*          | Transcription in specific languages <br/> LEN -> English, LDE -> German, LES -> Spanish and so on |

[^1]: Currently not available in Appliances
