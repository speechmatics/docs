## Speechmatics Flow SDK

Flow is the Speechmatics voice agent API that allows you to build responsive, interactive, and intelligent speech-to-speech interactions to any product.

The voice SDK provides a simple and effective way to integrate natural and fluid conversational interactions, with the ability to handle interuptions, multiple speakers and the ability to understand different dialects and accents.
