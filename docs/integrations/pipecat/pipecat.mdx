# Pipecat

Pipecat is an open-source framework for building real-time voice and multimodal conversational agents. It orchestrates audio and video input/output, speech-to-text, large language models, and text-to-speech into a single, unified pipeline. 

SpeechmaticsSTTService enables real-time speech transcription using Speechmaticsâ€™ WebSocket API with partial and final results, speaker diarization, and end of utterance detection (VAD) for comprehensive conversation analysis.
When integrated with Pipecat, you can build real-time voice and multimodal conversational agents specifically tailored to your use case. 

## Quickstart

### Use cases

Pipecat is perfect for:

- **Voice AI**: Voice assistants, chatbots, and IVR systems
- **Translation**: Realtime translation of live events or media
- **Accessibility applications**: Screen readers and assistive technologies
- **Content creation**: Podcasts, dubbing, audiobooks, and voice-overs
- **Media production**: News broadcasts and automated announcements
