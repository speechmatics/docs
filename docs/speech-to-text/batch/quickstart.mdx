# Quickstart

:::tip
The quickest way to try transcribing for free is by creating a Speechmatics account and using our [web portal](https://portal.speechmatics.com/jobs/create/batch).
<br/>
Set up your transcription job and click the <Button mx="1" size="1" color="gray" variant="soft"><CodeIcon size="8" />View config</Button> button on the output page to get the code and use it in the API.
:::

## Using the Batch API

### 1. Create an API key

[Create an API key in the portal here](https://portal.speechmatics.com/settings/api-keys), which you'll use to securely access the API.
Store the key as a managed secret.

:::info
Enterprise customers may need to speak to [Support](https://support.speechmatics.com) to get your API keys.
:::

### 2. Pick and install a library

Check out our [Batch Python client](https://github.com/speechmatics/speechmatics-python-sdk) or [JavaScript client](https://www.npmjs.com/package/@speechmatics/batch-client) to get started.

<Tabs>
  <TabItem value="python" label="Python">
  Install using pip:
  ```
  pip install speechmatics-batch
  ```
  </TabItem>
  <TabItem value="javascript" label="JavaScript">
  Install using NPM:
  ```
  npm install @speechmatics/batch-client
  ```
  </TabItem>
</Tabs>

### 3. Grab a sample file

Download and save our [example.wav](https://github.com/speechmatics/speechmatics-js-sdk/raw/7d219bfee9166736e6aa21598535a194387b84be/examples/nodejs/example.wav)

### 4. Run the example

Replace `YOUR_API_KEY` with your key, then run the script.

<Tabs>
  <TabItem value="python" label="Python">
  <CodeBlock language="python">
    {pythonQuickstart}
  </CodeBlock>
  You should see the following output:
  ```
  SPEAKER S1: Welcome to Speechmatics. We're delighted that you've decided to try our speech to text software to get going. Just create an API key and submit a transcription request to our API. We hope you'll be very impressed by the results. Thank you.
  ```
  </TabItem>
  <TabItem value="javascript" label="JavaScript">
  <CodeBlock language="javascript">
    {jsQuickstart}
  </CodeBlock>
  You should then see the following output:
  ```
  Sending file for transcription...
  Transcription finished!
  Welcome to Speechmatics . We're delighted that you've decided to try our speech to text software to get going . Just create an API key and submit a transcription request to our API . We hope you'll be very impressed by the results . Thank you .
  ```
  </TabItem>
</Tabs>
