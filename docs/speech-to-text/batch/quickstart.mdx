# Quickstart

Learn how to transcribe pre-recorded audio and video files.

:::tip
The quickest way to try transcribing for free is by creating a Speechmatics account and using our [web portal](https://portal.speechmatics.com/jobs/create/batch).
:::

## Using the Batch API

### 1. Create an API key

[Create an API key in the portal here](https://portal.speechmatics.com/settings/api-keys), which you'll use to securely access the API.
Store the key as a managed secret.

:::info
Enterprise customers may need to speak to [Support](https://support.speechmatics.com) to get your API keys.
:::

### 2. Pick and install a library

Check out our [Javascript client](https://www.npmjs.com/package/@speechmatics/batch-client) or [Python client](https://pypi.org/project/speechmatics-python/) to get started.

<Tabs>
  <TabItem value="javascript" label="Javascript">
  Install using NPM:
  ```
  npm install @speechmatics/batch-client
  ```
  </TabItem>
  <TabItem value="python" label="Python">
  Install using pip:
  ```
  pip3 install speechmatics-python
  ```
  </TabItem>
</Tabs>

### 3. Grab a sample file

Download and save our [example.wav](https://github.com/speechmatics/speechmatics-js-sdk/raw/7d219bfee9166736e6aa21598535a194387b84be/examples/nodejs/example.wav)

### 4. Insert API key

Paste your API key into `YOUR_API_KEY` in the code below.
